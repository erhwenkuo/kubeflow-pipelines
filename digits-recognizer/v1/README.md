
## Install packages

```bash
$ git clone https://github.com/erhwenkuo/kubeflow-pipelines.git
$ cd kubeflow-pipelines/digits-recognizer/v1
$ pip install -r requirements.txt
```

Checking result:

```bash
$  pip list | grep kfp
kfp                          1.8.22
kfp-pipeline-spec            0.1.16
kfp-server-api               1.8.5
```
